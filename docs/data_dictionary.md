# Data Dictionary: Singapore Heat Stress Analysis

## Source
- **API:** data.gov.sg Air Temperature dataset
- **Agency:** National Environment Agency (NEA)
- **Format:** JSON via REST API

## Field Descriptions

| Field Name | Data Type | Description | Example |
|------------|-----------|-------------|---------|
| id | SERIAL | Primary key (auto-generated by PostgreSQL) | 1 |
| timestamp | TIMESTAMP | Time of reading (Singapore Time SGT) | 2025-10-01 08:30:00 |
| station_id | VARCHAR(10) | Unique code for the weather station | S104 |
| station_name | VARCHAR(100) | Name of the location | Clementi |
| latitude | REAL | Geographic latitude (WGS84) | 1.315 |
| longitude | REAL | Geographic longitude (WGS84) | 103.765 |
| readings | REAL | Air temperature in degrees Celsius | 28.5 |
| date | DATE | Extracted date from timestamp | 2025-10-01 |
| month | INTEGER | Extracted month (1-12) | 10 |
| hour | INTEGER | Extracted hour (0-23) | 8 |

## Data Quality Notes

- Temperature range validated: 23°C - 36°C (typical for Singapore)
- Duplicate records removed
- Records with missing critical fields (station_id, temperature, timestamp) were excluded
- Some timestamps may have no data due to API limitations
